Declaração de variáveis

Variáveis servem para guardar dados.

Declarando variáveis:

            var n1 = 1;
            var n2 = 2;
            var n3 = 3;

Declarando strings:

            var s1 = "JavaScript";
            var s2 = 'Curso em Vídeo';
            var s3 = `Guanabara`;

Tipos primitivos primordiais

  • number
    • Infinity
    • NaN (not a number)
  • String
  • Boolean
  • null
  • underfined
  • object
    • Array
  • function

typeof é um comando para saber o tipo de uma variável

Regras para identificação de variáveis

  • Podem começar com letra, $, ou _
  • Não podem começar com números
  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas (Palavaras usadas em comandos), Ex.: var

Dicas

  • Maiúsculas e minúsculas fazem diferença
  • Tente escolher nomes coerentes para variáveis
  • Evite see tornar um 'programador alfabeto' (var a, b, c ...) ou um 'programador contador' (var n1, n2, n3...)
