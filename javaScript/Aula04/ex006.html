    • ==
      1. Caso de identidade
        1. ↪ = (recebe)
        2. ↪ == (O sinal de igualdade testa a grandeza/valor e não o tipo)
          • 5 == '5' → true
        3. ↪ === (O operador de identidade ou igualdade restrita. Testa se os valores são idênticos em valor e tipo)
          • 5 === '5' → false
          • 5 === 5 → true

  • Ternários
    • Ele se chama Ternários pois tem três partes. Teste ? true : false
      • ?
      • :
    • Teste→ Nessa primeira parte vai um teste lógico e irá resultar um true ou false
    • True→ O que vai acontecer se for true
    • False→ O que vai acontecer se for false
      • Ex.: media >= 7.0 ? "Aprovado" : "Reprovado"
