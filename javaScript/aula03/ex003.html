Lidando com dados no prompt

Falando de JavaScript nós temos de aprender a lidar com o "+".
O "+" no js tem dois sentidos de uso, adição e concatenação. E quando usar os dois?
number + number = adição
String + String = concatenação

o comando window.prompt retorna uma string, logo, para converter o valor em um number
podemos usar os comandos: 

        
            Number.parseInt(x) //valor inteiro
            Number.parseFloat(x) //valor real
            ou apenas
            Number(n)
A conversão também se é aplicada para String: 
            
                String(x)
                ou
                x.toString()
